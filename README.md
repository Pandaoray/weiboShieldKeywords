## 微博屏蔽词增强助手
## 概述
为微博屏蔽词提供一键式管理 ———— 一键续期、一键导入、一键导出、一键清空。 注意：受限于微博功能限制，非会员无法使用屏蔽词功能。

微博会员的「屏蔽关键词」功能单一，关键词有效期只有30天，到期后自动解除。

该插件即为解决此限制所作，在屏蔽关键词页面中增加了「一键续命」按钮，点击一下即可重置所有关键词的有效期。

另外，目前已支持「一键导出」「一键导入」「一键清空」。
